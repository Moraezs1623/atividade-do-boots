Dicas e Curiosidades

Aqui você encontra dicas úteis e curiosidades interessantes.

Dica do Dia

Você sabia que o Bootstrap foi criado por desenvolvedores do Twitter para melhorar a consistência no design? Hoje, é um dos frameworks mais populares para desenvolvimento web.

Curiosidade

O nome "Bootstrap" vem do conceito de "bootstrap" que significa levantar algo a partir de um estado básico, assim como o framework levanta a qualidade do design da web a partir de um ponto básico.

Sugestão

Experimente utilizar as classes de utilitários do Bootstrap para ajustar rapidamente o layout e o design sem a necessidade de escrever CSS adicional.
